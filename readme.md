### Cценарій використання Google Sheets API
Ми можемо використовувати таблички від Google в багатьох сферах. Наприклад в якості дуже примітивної безкоштовної бази даних. Розробивши відповідний функціонал CRUD операцій та клієнтську частину. 
Наразі можемо отримувати дані і використовувати їх у своєму додатку. 
Наприклад якщо надалі будемо будувати додаток використовуючи SPA (single page app) на стороні клієнта, то клієнт може робити запит на отримання цієї інформації в належному провалідованому вигляді для рендеру.
Або ж якщо вирішимо використовувати MVP (model view controller) і сервер сам буде робити рендер інформації, а не просто віддавати її, то відповідно зможемо передати отриманні дані з таблиці в template файл (view) і там виводити. 


### Приклад застосування API
Надсилаючи запит на /sheets/:sheetName , вказавши замість sheetName назву таблички, отримаємо дані цієї таблиці.
